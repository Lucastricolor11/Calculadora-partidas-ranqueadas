# Calculadora de Partidas Rankeadas

Projeto desenvolvido para o desafio da DIO.

## Objetivo

Criar uma função que calcula o saldo de vitórias de um jogador e define seu nível de rank.

## Tecnologias

- JavaScript

## Regras

- Ferro: até 10 vitórias
- Bronze: 11 a 20 vitórias
- Prata: 21 a 50 vitórias
- Ouro: 51 a 80 vitórias
- Diamante: 81 a 90 vitórias
- Lendário: 91 a 100 vitórias
- Imortal: 101 ou mais vitórias

## Exemplo

O Herói tem saldo de 65 e está no nível de Diamante.
